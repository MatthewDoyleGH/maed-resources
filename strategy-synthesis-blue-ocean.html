Blue Ocean Strategy Synthesis Process

How value-innovation frameworks combine to produce uncontested market space

From Red Ocean to Blue Ocean

Blue ocean strategy is not about out-competing rivals in an existing industry — it is about making rivals irrelevant by creating uncontested market space. This map traces how boundary-crossing demand scans and current-state value mapping are progressively synthesised into a new value curve that delivers higher buyer value at lower cost.

Stage 3
Value Innovation Synthesis
Breaking the cost-versus-value trade-off
V
Value Innovation Map
Where low cost and high buyer value can coexist
The convergence point. The Six Paths scan shows where new buyer demand lives; the As-Is Canvas shows where current effort is wasted or commoditised. Plotting the two together exposes the red-ocean trade-off that conventional strategy accepts — and pinpoints where it can be broken by simultaneously lowering cost and lifting buyer value.
Blue Ocean
High buyer value AND low cost structure — the uncontested space where value innovation breaks the conventional cost-vs-differentiation trade-off
Dying Middle
Low buyer value AND high cost — unsustainable: being out-competed on both dimensions at once
Premium Red Ocean
High buyer value, high cost — defensible only until a value-innovator arrives with equivalent value at a fraction of the cost
Commodity Red Ocean
Low buyer value, low cost — a shrinking price-war margin where every competitor looks identical on the canvas

4
Four Actions Framework
Eliminate, Reduce, Raise, Create — the levers of value innovation
The action engine of blue ocean strategy. Having identified where value innovation is possible, the Four Actions Framework forces the organisation to answer four questions about the existing industry value curve. Together, the answers define a new curve that lowers cost AND lifts buyer value at the same time — the trade-off that conventional strategy says is impossible.

The Synthesis Principle
Six Paths is a boundary-crossing tool. The As-Is Canvas is a reality check. The Value Innovation Map is the synthesis — the point at which "different" and "cheaper" stop being a trade-off. The Four Actions Framework is the execution discipline. Each link in the chain asks a different question, and skipping any one collapses blue ocean strategy back into either me-too innovation or unsupported cost-cutting.
